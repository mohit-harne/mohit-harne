# 💫 About Me:
Full Stack Developer with 3+ years of experience in building scalable web applications using React.js, Next.js, Node.js, and Express.js. Strong frontend expertise in developing responsive UI with Tailwind CSS, JavaScript (ES6+), and reusable components. Experienced in REST API development and integration, state management using Redux Toolkit, and handling asynchronous data. Proficient in MongoDB and PostgreSQL for database design and CRUD operations. Worked on enterprise applications including admin dashboards, management systems, and real-time features. Skilled in performance optimization, code splitting, lazy loading, and API optimization. Familiar with authentication (JWT), Git, Agile methodology, and deployment on Vercel. Currently learning TypeScript, advanced Next.js (App Router, Server Actions), Docker, and cloud concepts.
